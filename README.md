# c-netprog-exercise-blue-pizza
Blue Pizza delivery client and server program in C

## Usage
### Server
```
$ gcc blue-pizza-server.c -o server
# ./server
usage: ./server <port>
```
### Client
```
$ gcc blue-pizza-client.c -o client
$ ./client
usage: ./client <port> <hostname/IP>
```
